# Image Colorizer
**Final Year B.Tech IT Project — Automatic Photograph Colorization using Deep Learning**

---

## What This Project Does
This system automatically colorizes black & white photographs using three AI models:
1. **Pix2Pix GAN** — custom U-Net architecture, trained from scratch on COCO 2017 (200 epochs)
2. **ResNet GAN** — ResNet34 transfer learning, faster convergence, best overall metrics (100 epochs)
3. **OpenCV CNN** — Zhang et al. 2016 classic pretrained colorization model (baseline)

Users upload a grayscale photo via the web app and receive a colourized result in ~1 second.
The system automatically detects portrait vs landscape and loads the best-performing checkpoint.

## Smart Checkpoint System

Uses OpenCV face detection to auto-select the best checkpoint for each image:
- **Portrait** (face detected) → ResNet ep.31 (SSIM 0.9021, PSNR 23.40 dB)
- **Landscape** (no face) → GAN ep.131 (SSIM 0.9219, PSNR 23.41 dB)
- Falls back to aspect ratio if OpenCV unavailable (height > width × 1.1 = portrait)
- User can override by clicking GAN / ResNet / OpenCV buttons in the web UI header

`best_checkpoints.json` stores all 4 keys (`gan_portrait`, `gan_landscape`,
`resnet_portrait`, `resnet_landscape`). Switching models in the UI always uses
the correct scene-specific checkpoint for that model.

To regenerate after adding new checkpoints:

    python checkpoint_picker.py              # test all checkpoints, 20 images per type
    python checkpoint_picker.py --limit 5   # faster, 5 images per type

## Training History (Full Record)

| Session | GAN Epoch | GAN PSNR | ResNet Epoch | ResNet PSNR | ResNet SSIM |
|---|---|---|---|---|---|
| Phase 1 complete | 100 | 25.85 dB* | — | — | — |
| Phase 2 start | 130 | 22.60 dB | 30 | 23.18 dB | 0.8846 |
| LR dip (expected) | 147 | 22.30 dB | 42 | 22.71 dB | 0.8890 |
| Recovery | 170 | 22.71 dB | 59 | 23.39 dB | 0.8971 |
| | 177 | 22.49 dB | 65 | 23.28 dB | 0.8987 |
| | 185 | 22.28 dB | 71 | 22.14 dB | 0.8785 |
| **FINAL** | **200** | **22.41 dB** | **100** | **22.49 dB** | **0.8929** |

*Phase 1 PSNR measured on 5k-image subset, not directly comparable to Phase 2 (118k images).

**Why ResNet ep.31 beats ep.100 for portraits:**
Transfer learning peaks early for specific content types. The pretrained ResNet34 encoder
already understands face features from ImageNet. Later epochs trade portrait accuracy for
general scene diversity — standard behaviour with pretrained feature extractors.

**Why LR dips cause temporary drops (ep.42, ep.83):**
StepLR halves learning rate every 30 epochs. First few epochs after each halving show
instability as the optimizer adjusts. Recovery takes 6–10 epochs and is expected.

## API Reference

    GET /health
    Returns: { model_ready, active_model, device, gan_available, resnet_available,
               opencv_available, gan_path, resnet_path, smart_config }

    POST /switch-model
    Body:    { "model": "gan" | "resnet" | "opencv" }
    Returns: { "success": true, "active": "resnet", "model": "generator_resnet_epoch100" }

    POST /colorize
    Body:    { "image": "data:image/jpeg;base64,...", "color_boost": 1.4,
               "saturation": 1.0, "contrast": 1.0, "sharpness": 1.0 }
    Returns: { "success": true, "colorized": "data:image/jpeg;base64,...",
               "time": 1.2, "psnr": 23.40, "ssim": 0.9021,
               "active_model": "resnet", "checkpoint": "generator_resnet_epoch31",
               "image_type": "portrait" }

    POST /batch
    Body:    { "images": ["data:image/jpeg;base64,...", ...], "color_boost": 1.4 }
    Returns: { "results": [{ "index": 0, "success": true, "colorized": "...",
                              "psnr": 23.40, "ssim": 0.9021 }] }

## Key Design Decisions (Viva Reference)

| Decision | Reasoning |
|---|---|
| CIE-LAB colour space | L (lightness) preserved exactly; model only predicts AB (colour). Avoids brightness errors. |
| 512×512 inference | Higher than typical 256×256 — better colour boundary definition at object edges. |
| Smart per-scene boost | Portrait=1.0 (natural skin), Landscape=1.4 (vivid sky/grass). Reduces orange face artefacts. |
| SSIM as ranking metric | More perceptually relevant than PSNR for colorization; penalises structural artefacts. |
| Two-phase GAN training | Phase 1 on 5k images: fast initial convergence. Phase 2 on 118k: real-world diversity. |
| ResNet transfer learning | ImageNet-pretrained features → better generalisation with 100 vs 200 GAN epochs. |
| colour_boost=1.0 for portraits | Prevents orange/green skin tones — known GAN colorization artefact. |
| 303 checkpoints tested | Data-driven selection beats intuitive "use the last epoch" approach. |
